## 디렉토리 개요
레포 루트 기준 `.agents/skills`는 Codex용 스킬 복사본 디렉토리다. 원천은 `skills/`이며, 이 경로는 런타임 안정성을 위해 심볼릭 링크 대신 실파일 복사로 유지한다.

## 파일 작성 컨벤션
- 원천과 동일한 디렉토리 구조/파일명을 유지한다.
- 각 스킬 디렉토리에 `SKILL.md`가 존재해야 한다.
- 이 디렉토리에 신규 스킬을 직접 만들지 않는다.
- 원천 갱신 후 전체 복사로 동기화하며, 부분 수동 수정은 금지한다.

## 코드 작성 컨벤션
- 원천(`skills/`)과 내용이 달라지면 원천을 기준으로 즉시 덮어쓴다.
- 이 디렉토리의 파일은 실행용 아티팩트로 간주하고, 리뷰 시에도 원천 변경 여부를 우선 확인한다.
- Codex에서 스킬 미탐지 문제가 다시 발생하면 먼저 세션 재시작을 수행하고, 이후 복사 상태를 검증한다.
