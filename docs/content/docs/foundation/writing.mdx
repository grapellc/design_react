## Familiar

**익숙한 단어를 사용**해요. **기술적인 용어나 한자어**는 **지양**해요.

| ✅ Do                           | ❌ Don't                         |
| ------------------------------- | -------------------------------- |
| 상대방과 대화할 수 없어요.      | 상대방과 대화가 불가능합니다.    |
| 동네인증을 할 수 없는 위치예요. | 동네인증 가능한 위치가 아니에요. |

에러 메시지에는 **직접 해결할 수 있는 방법**을 알려주고, **해결 방법**도 함께 알려주세요.

| ✅ Do                                                                        | ❌ Don't                                         |
| ---------------------------------------------------------------------------- | ------------------------------------------------ |
| 죄송합니다. 일시적으로 서비스를 사용할 수 없어요. 잠시 후 다시 시도해주세요. | 서버 에러 / unknown error 등 [장애상황/기술용어] |

## Honorifics

**존칭어 사용을 최소화**해요. 정말 꼭 필요할 때만 존칭어를 사용하세요.

| ✅ Do                                               | ❌ Don't                                     |
| --------------------------------------------------- | -------------------------------------------- |
| 최대 2개의 글을 등록할 수 있어요.                   | 최대 2개의 글을 등록하실 수 있어요. [존칭어] |
| 혹시 자전거를 찾고 있나요?                          | 혹시 자전거를 찾고 계신가요? [존칭어]        |
| 보리엄마님, 꿈돌이님과 거래 어떠셨나요?             | -                                            |
| 보리엄마님, 사고 싶은 물품의 키워드를 등록해보세요. | -                                            |

유저의 이름을 부를 때는 조금 더 고민해봐요. 소리내어 읽어보면 좋아요.

## Numbers

수를 표현할 때는 **아라비아 숫자를 사용**해요.

| ✅ Do                             | ❌ Don't                                   |
| --------------------------------- | ------------------------------------------ |
| 보리엄마님이 당근 1개를 보냈어요. | 보리엄마님이 당근 한 개를 보냈어요. [숫자] |

## Abbreviations

**축약어, 은어, 속어, 유행어** 등은 **지양**해요.

| ✅ Do                                   | ❌ Don't                                                   |
| --------------------------------------- | ---------------------------------------------------------- |
| 끌어올리기는 매일 5회까지 할 수 있어요. | 끌올은 사용자마다 하루에 최대 5회까지 가능합니다. [축약어] |

사용자들이 끌올이라고 줄여부르더라도 우리는 끌어올리기라고 이야기해야 해요.

## Function Names

기능의 이름보다는 **기능의 목적**을 전달해요.

| ✅ Do                                      | ❌ Don't                                            |
| ------------------------------------------ | --------------------------------------------------- |
| 보리엄마님이 참외죠아님의 댓글을 좋아해요. | 보리엄마님이 참외죠아님의 댓글에 좋아요를 눌렀어요. |

## Positive Sentences

같은 내용이라면, **긍정문**을 사용해보세요.

| ✅ Do                                             | ❌ Don't                                            |
| ------------------------------------------------- | --------------------------------------------------- |
| 닉네임은 한 달에 한 번만 변경할 수 있어요. [긍정] | 한 달에 한 번 이상 닉네임을 변경할 수 없어요.[부정] |

## Active Sentences

**능동문**을 사용해요.

| ✅ Do                                                      | ❌ Don't                                                                                                             |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| 잠깐만요! (보리엄마님에게만 보이는 메시지입니다.) [피동문] | 잠깐만요! (보리엄마님에게만 보여지는 메시지입니다.) [이중피동], 잠깐만요! (보리엄마님만 보는 메시지입니다.) [능동문] |

1. 가이드보다는 좋은 문장이 먼저예요. 능동문보다 피동문이 의미 전달이 더 잘 된다면? 당연히 피동문을 사용해요.
2. 자동으로 발송되는 메시지고, 사용자가 조심해야하는 상황에 나오는 메시지라면 '~요' 대신 **'~니다'** 를 사용하여 문장에 무게를 실어요.

## User Actions

처리, 완료 등 데이터의 변화보다는 **사용자의 행위**를 표현하는 말에 집중해보세요.

| ✅ Do                                            | ❌ Don't                                                |
| ------------------------------------------------ | ------------------------------------------------------- |
| 거래 후기를 보냈어요.                            | 거래 후기 작성을 완료했어요.                            |
| 보리엄마님의 게시글 모두 2일 뒤에 숨겨질 거예요. | 보리엄마님이 등록하신 게시물은 이틀 후에 숨김 처리돼요. |

## Emotions

사용자의 **마음에 충분히 와닿았으면 해요**. 진심을 담아 **사려 깊게 이야기**해요.

| ✅ Do                                                                                                                   | ❌ Don't                                                         |
| ----------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| 채팅으로 거래하기 [기능의 목적]                                                                                         | 채팅하기 [기능의 이름]                                           |
| 잠깐만요, 혹시 지금 대화가 불쾌하지는 않으셨나요? 상대방이 거래와 관련 없는 대화를 시도하면, 당근 팀에게 신고해주세요!  | 혹시 상대방이 이상한 대화를 유도하나요?                          |
| 거래 약속을 깜박하지는 않으셨나요? 약속을 잘 지키는 따뜻한 이웃이 되어보세요.                                           | 혹시 약속을 자꾸 잊는다면 거래 약속 알림을 사용해보는 건 어때요? |
| 주소가 공유되었네요! 당근마켓은 택배거래보다 직거래를 권장해요. 직거래할 때는 누구나 찾기 쉽고 안전한 공공장소가 좋아요 | -                                                                |
| 슬기맘님은 지금 첫 거래를 하고 있어요. 당근마켓 이용이 다소 서툴터라도 따뜻하게 환영해주세요.                           | -                                                                |

## Focus on One Purpose

**하나의 목적에 집중**해요. 목적을 해치는 부가적인 설명은 최소화해요.

- 더 설명하고 싶은 당신의 마음을 이해해요. 하지만 긴 문장을 읽기 힘들 수 있어요.
- 혹시 문장이 길어지고 있나요? 문장 하나에는 하나의 내용만 담고 문장을 분리해보세요.

## Local Community

사는 곳이나 **지역사회**, **지역사회의 구성원**의 이야기를 담아보세요.
가입 권유 푸시 메시지에도 사용자의 동네 이름을 넣어요. 당근이라는 공간은 우리 동네를 연결하는 커뮤니티라는 걸 꾸준히 알려주려 해요.

| ✅ Do                                                                                                             | ❌ Don't |
| ----------------------------------------------------------------------------------------------------------------- | -------- |
| 지금 역삼동 근처 주민들과 중고 직거래를 시작해보세요. 내가 쓰지 않는 물건이 누군가에게는 필요한 물건일 수 있어요! | -        |

## Exclamation Marks

**느낌표**는 **꼭 필요한 순간에 사용**해요. 반복적인 느낌표는 의미를 잃어요.

## Spaces

**띄어쓰기**는 기능명, 서비스명, 상태값 자체를 의미할 때는 붙여써요.

| ✅ Do                                                              | ❌ Don't                                                            |
| ------------------------------------------------------------------ | ------------------------------------------------------------------- |
| 예약중                                                             | 예약 중                                                             |
| 거래를 약속하셨나요? "티슈 케이스.." 글을 [예약중]으로 바꿔보세요. | 거래를 약속하셨나요? "티슈 케이스.." 글을 [예약 중]으로 바꿔보세요. |
| 예약 중일 때는 채팅할 수 없어요.                                   | 예약중일 때는 채팅할 수 없어요.                                     |

## Periods

**마침표**는 **평서문이나 명령문의 마지막에 사용**해요.

단, **아래의 경우에는 마침표를 사용하지 않아요**.

- 메인 타이틀 영역 문구
- 큰 글씨(20pt 이상)로 들어가는 문구
- 버튼, 메뉴, 라벨에 들어간 문구
- 버튼에서 문장을 사용하고 긍정 혹은 부정의 감탄사(네, 아니요)가 있다면 쉼표를 넣어주세요.
