당근의 아이콘은 당근 로고의 형태적 특징을 바탕으로 친근하고 부드러운 인상을 전달합니다.
명확한 메타포와 직관적이고 간결한 형태, 굵은 선을 사용해 누구나 쉽게 이해할 수 있도록 제작되었습니다.
이러한 당근의 아이콘은 글로벌 사용을 고려하여 모두가 공감할 수 있는 디자인을 목표로 합니다.

## Creation Guide

당근의 모노크롬, 멀티컬러 아이콘은 총 300여개에 달하며, UI에 필요한 대부분의 아이콘이 일관된 스타일과 형태 규칙에 맞춰 완성도있게 제작되어 있습니다.
가능한 현재 제작된 범위 내에서 사용하는 것을 권장드립니다. 아이콘 라이브러리에서 적합한 아이콘을 찾아 활용해 보세요.

- [Icon Library](/docs/foundation/iconography/library)
- Figma 파일 참고

그럼에도 불구하고 새로운 아이콘 제작이 필요한 경우, 아래 절차를 따라주세요.

1. 필요한 아이콘이 모노크롬과 멀티컬러 중 어떤 아이콘인지 검토합니다.
2. 제품팀에서 적절한 메타포를 선정하고 초안을 그립니다.
3. 필수 디자인 코어팀에 형태 검토를 요청합니다. 상세 가이드에 맞춰서 다듬습니다.
4. 앱에 공용으로 사용될만한 아이콘이라면, 디자인 코어팀에서 아이콘 라이브러리에 등록합니다.
