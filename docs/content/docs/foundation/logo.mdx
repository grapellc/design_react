당근의 로고는 서비스 전체를 상징하는 소중한 브랜드 자산이에요.
다양한 매체에서 당근의 브랜드 아이덴티티를 올바르게 표현하고 브랜드 이미지를 일관성 있게 유지하기 위해, 아래 가이드가 규정하는 범위 내에서 로고를 적용해 주세요.

## 시그니처 로고 (Primary)

당근의 시그니처 로고는 심볼과 워드마크가 결합된 형태입니다. 시그니처 활용 시 하단에 명시된 형태와 규정에 맞게 사용해야 하며,
로고의 시각적 독립성을 위해 규정된 주변의 최소 공간(Clear Space)을 다른 매체나 그래픽 요소가 침범하지 않도록 주의하여 사용해 주세요.

## 심볼 로고 (Secondary)

당근의 심볼 로고입니다. 대부분의 경우 시그니처 로고를 활용할 것을 권장하나, 당근이 주체임이 명확한 때에 한하여 디자인 표현의 다양성을 위해 심볼의 단독 사용을 고려할 수 있어요.

심볼 활용 시 하단에 명시된 형태와 규정에 맞게 사용해야 하며, 로고의 시각적 독립성을 위해 규정된 주변의 최소 공간(Clear Space)을 다른 매체나 그래픽 요소가 침범하지 않도록 주의하여 사용해 주세요.

## 앱 아이콘

심볼을 앱 아이콘 형태로 커뮤니케이션 할 수 있어요. 지정된 크기와 여백 변형 없이 사용해 주세요.

## 사용 가이드

다양한 매체에서 브랜드 아이덴티티를 올바르게 표현하고 브랜드 이미지를 일관성 있게 유지하기 위해, 아래 가이드가 규정하는 범위 내에서 로고를 적용해 주세요.

### 컬러 (Color)

채소 당근을 연상시키는 따뜻하고 환한 주황색은 당근마켓을 대표하는 브랜드 컬러입니다.

<Image src="/docs/foundation/logo/color-guide.webp" alt="컬러 가이드" />

### 로고의 배경색

로고의 당근색이 가장 잘 드러나도록 흰색 또는 밝은 배경에서 사용할 것을 권장하며,
브랜드 컬러에 포함되지 않는 제 3의 색을 합당한 이유 없이 사용하지 않아요.
컬러 로고를 사용할 수 없을 때에는 브랜딩 팀과의 협의 후 단색 로고를 사용할 수 있어요.

### 유의사항 (Usage Guide)

브랜드 이미지가 손상되지 않도록 아래의 예시를 참고하여 반드시 표준 형태를 사용해 주세요. 이 외 판단이 어려운 경우 당근 브랜딩 팀으로 문의해 주세요.

<Image src="/docs/foundation/logo/usage-guide.webp" alt="유의사항 가이드" />

### 사용 허가에 관하여 (Brand Permission)

아래에 해당되는 경우에는 브랜드 리소스 사용을 허가하지 않아요.

1. 협의되지 않은 마케팅과 상업 목적의 제작물 일체
2. 개인적/상업적 목적 관계 없이 협의되지 않은 양산형 제품과 패키지에 프린트

당근과의 공식적인 협의 하에 로고를 활용하는 경우, 최종 버전의 제작물을 당근 브랜딩 팀의 담당자에게 전달하여 브랜드 가이드 준수 여부를 확인하세요.
