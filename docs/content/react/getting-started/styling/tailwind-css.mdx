SEED에서 제공하는 토큰을 Tailwind CSS 3과 4 유틸리티 클래스를 통해 쉽게 사용할 수 있습니다. 사용하는 Tailwind 버전에 맞는 패키지를 선택하세요.

<Tabs items={['Tailwind CSS 4', 'Tailwind CSS 3']}>

<Tab value="Tailwind CSS 4">

<Steps>
<Step>

### 의존성 설치

```package-install
@grape_design_react/css @grape_design_react/tailwind4-theme
```

</Step>

<Step>

### base.css import하기

```ts title="index.tsx"
import "@grape_design_react/css/base.css";
import "./style.css"; // 다음 단계 참고
```

</Step>

<Step>

### CSS 파일 설정하기

```css title="style.css"
@import "tailwindcss";
@import "@grape_design_react/tailwind4-theme";
```

</Step>

<Step>

### Theming 설정하기

Tailwind CSS 스타일시트는 theming 스크립트와 별개로 작동합니다. 아래 가이드를 참고하여 테마 설정을 진행하세요.

- [Vite](/react/getting-started/installation/vite)
- [Rsbuild](/react/getting-started/installation/rsbuild)
- [Webpack](/react/getting-started/installation/webpack)
- [Manual](/react/getting-started/installation/manual)

</Step>

</Steps>

</Tab>

<Tab value="Tailwind CSS 3">

<Steps>
<Step>

### 의존성 설치

```package-install
@grape_design_react/css @grape_design_react/tailwind3-plugin
```

</Step>

<Step>

### base.css import하기

```ts title="index.tsx"
import "@grape_design_react/css/base.css";
```

</Step>

<Step>

### Tailwind 설정 파일 수정하기

```js title="tailwind.config.js"
import seedPlugin from "@grape_design_react/tailwind3-plugin"; // [!code highlight]

/** @type {import('tailwindcss').Config} */
export default {
  content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [seedPlugin], // [!code highlight]
};
```

</Step>

<Step>

### Theming 설정하기

Tailwind CSS 스타일시트는 theming 스크립트와 별개로 작동합니다. 아래 가이드를 참고하여 테마 설정을 진행하세요.

- [Rsbuild](/react/getting-started/installation/rsbuild)
