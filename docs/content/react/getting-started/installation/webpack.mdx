---
title: Webpack
description: Webpack 프로젝트에서 Seed Design을 설치하고 사용하는 방법을 알아봅니다.
---

<Steps>
<Step>

### 의존성 설치

필요한 의존성들을 설치해요.

```package-install
@grape_design_react/react @grape_design_react/css @grape_design_react/webpack-plugin
```

</Step>

<Step>

### base.css import하기

```ts title="index.tsx"
import "@grape_design_react/css/base.css";
```

</Step>

<Step>

### Webpack config 설정하기

```ts title="vite.config.ts"
+import { SeedDesignPlugin } from '@grape_design_react/webpack-plugin';

export default defineConfig({
  plugins: [
+    new SeedDesignPlugin()
  ]
})
```

</Step>

<Step>

### grape_design_react.json 생성하기

<Tabs items={['CLI로 추가', '직접 추가']}>
<Tab value="CLI로 추가">
<Steps>
<Step>

### init 명령어 입력하기

</Step>
```sh
npx @grape_design_react/cli@latest init
```
<Step>

### grape_design_react 설정

```sh
◇  TypeScript를 사용중이신가요?
│  Yes
│
◇  React Server Components를 사용중이신가요?
│  No
│
◇  grape_design_react 폴더 경로를 입력해주세요. (기본값은 프로젝트 루트에 생성됩니다.)
│  ./grape_design_react
```

</Step>
</Steps>
</Tab>

<Tab value="직접 추가">

아래 파일을 프로젝트 루트에 생성해주세요.

```json title="grape_design_react.json"
{
  "rsc": false,
  "tsx": true,
  "path": "./grape_design_react"
}
```

</Tab>
</Tabs>

</Step>

<Step>

### 컴포넌트를 한번 사용해볼까요?

```sh
npx @grape_design_react/cli@latest add ui:action-button
```

</Step>

<Step>

### tsconfig.json 설정하기

</Step>

```json title="tsconfig.json"
{
  "compilerOptions": {
    // ... your options
    "paths": {
      // [!code highlight]
      "grape_design_react/*": [
        // [!code highlight]
        "./grape_design_react/*"
      ]
    }
  }
}
```

</Steps>
