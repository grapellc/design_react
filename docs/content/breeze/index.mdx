## 소개

Seed Breeze는 다양한 프로젝트에서 바로 사용할 수 있는 유용한 UI 컴포넌트들을 모아놓은 큐레이션입니다.
각 컴포넌트는 독립적으로 작동하며, 특정 디자인 시스템에 종속되지 않도록 설계되었습니다.
필요한 컴포넌트의 코드를 복사하여 프로젝트에 맞게 커스터마이징할 수 있습니다.

## 특징

- 🚀 **즉시 사용 가능**: 복사하여 바로 사용할 수 있는 완성된 컴포넌트
- 🔧 **완전한 커스터마이징**: 소스 코드를 직접 수정하여 프로젝트에 맞게 조정
- 📦 **최소 의존성**: 필수 라이브러리만 사용하여 가볍게 유지

## 사용 방법

1. 원하는 컴포넌트 선택
2. Manual Installation을 통해 의존성 설치
3. 컴포넌트와 스타일 코드를 프로젝트에 복사
4. 필요에 따라 스타일과 기능 커스터마이징
