## 소개

SEED Design은 대규모 언어 모델(LLM)이 문서를 쉽게 이해하고 활용할 수 있도록 [llms.txt](https://llmstxt.org/) 형식의 문서를 제공합니다.
각 섹션별로 구조화된 텍스트 파일을 통해 AI 도구가 SEED Design의 컴포넌트, 가이드라인, API를 효과적으로 참조할 수 있습니다.

## 빠른 접근

자주 사용되는 문서에 빠르게 접근할 수 있습니다.

- **전체 React 문서**: [/react/llms.txt](https://grape_design_react.io/react/llms.txt) - React 라이브러리의 모든 문서를 하나의 파일로 제공
- **컴포넌트 가이드라인**: [/docs/llms.txt](https://grape_design_react.io/docs/llms.txt) - 디자인 가이드라인 문서 진입점

## 사용 예시

### AI 어시스턴트에게 컨텍스트 제공

```
@https://grape_design_react.io/react/llms-full.txt

SEED Design React 라이브러리를 사용해서 버튼 컴포넌트를 구현해주세요.
```

### 특정 컴포넌트 문서 참조

```
@https://grape_design_react.io/react/llms-components/action-button.txt

ActionButton 컴포넌트의 variant 옵션에 대해 설명해주세요.
```
