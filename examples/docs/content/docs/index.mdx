Welcome to **Grape Design** – a design system that mirrors [Seed Design](https://seed-design.io/react) components, built for React and React Native.

## Structure

- **Foundation**: Layout and typography primitives (Box, Flex, Grid, HStack, VStack, Text, etc.)
- **Components**: UI components (Action Button, Badge, Bottom Sheet, and more)

Each component page includes live previews using the same examples as the Vite showcase.
