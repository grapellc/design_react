# Vite

<Steps>
<Step>

### Install dependencies

```bash
pnpm add @grapu-design/react @grapu-design/css
```

</Step>

<Step>

### Import base CSS

```ts title="main.tsx"
import "@grapu-design/react/styles.css";
```

</Step>

<Step>

### Use components

Import and use components from `@grapu-design/react`.

```tsx
import { ActionButton } from "@grapu-design/react";

export function App() {
  return <ActionButton>Click me</ActionButton>;
}
```

</Step>
</Steps>
