# CLI commands

When available, use the Grape Design CLI to add components to your project.

Check the repository for the latest CLI usage. Typical commands include:

- **init** — Set up Grape Design in the project
- **add** — Add a component or snippet

See [Configuration](/react/getting-started/cli/configuration) for CLI config options.
