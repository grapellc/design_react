# Available transforms

Check the repository for the current list of codemod transforms. Common transforms may include:

- **replace-react-icon** — Replace React icon imports with the design system iconography
- **migrate-tokens** — Update design token names across the codebase

Run `npx @seed-design/codemod@latest --list` to see available transforms and usage.
