# Codemods

Use codemods to automate code changes when upgrading or migrating.

## Usage

Run a transform (when available):

```bash
npx @seed-design/codemod@latest <transform> <target_path> [options]
```

List available transforms:

```bash
npx @seed-design/codemod@latest --list
```

<Card title="Available Transforms" href="/react/developer-tools/codemods/available-transforms">

See the list of available transforms.

</Card>

## Options

- `--list` — Show available transforms
- `--log` — Enable verbose logging

Check the repository for the latest CLI and Node.js version requirements.
