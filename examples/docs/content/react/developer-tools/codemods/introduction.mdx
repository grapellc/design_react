# Codemods

Use codemods to automate code changes when upgrading or migrating.

## Usage

Grape Design does not ship a codemod package. For automated migrations you can refer to [Seed Design's codemod](https://github.com/daangn/seed-design) or apply code changes manually when upgrading.
